# xendit-payment-rn

Android and iOS wrapper for XenditPayment SDK

## Installation

```sh
npm install xendit-payment-rn
```

## Usage


```js
import { multiply } from 'xendit-payment-rn';

// ...

const result = await multiply(3, 7);
```
